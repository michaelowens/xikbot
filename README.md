# xikbot

> A highly configurable and extensible Twitch bot written in Node.js.

## Build Setup

``` bash
# install dependencies
npm install

# compile web assets
gulp build

# watcher (for compiling web assets while developing)
gulp watch

# run both bot & web interface
node index.js both

# run bot only
node index.js bot

# run web only
node index.js web

```
